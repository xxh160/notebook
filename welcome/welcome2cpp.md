## p pp ppp

`c++`最具特色的就是其指针和内存管理机制，如果不是这两个，谁没事用`c++`呀。

那么让我们体验一下这两点的迷人之处吧！

请写出一下程序的输出：

```c++
int n = 4;
char **a;
a = (char**) malloc (n * n * sizeof(char));
for (int i = 0; i < n; ++i) 
    for (int j = 0; j < n; ++j) { 
        printf("%p ", *a + n * i + j);
        printf("%p\n", *(a + i) + j);
    }
```

`malloc`一段连续内存然后强转成二级指针是无法形成二维数组的，实际写程序的时候千万别这么写。

提示：对于空指针，`%p`的输出会是`(nil)`。

## type

类型是什么？

对于`c/c++`来说，类型只是解释数据的一种方式。

一段内存中的数据，比如`00110000`，如果它是`char`，那它就是`'0'`；如果它是`int`，那它就是`48`。

所以很多时候，我们可以通过转换类型来得到一些靠平常手段难以得到的信息。

那么，请写出一下程序的输出：

```c++
int a = (1 << 2) + (1 << 8) + (1 << 9) + (1 << 17) + (1 << 24);
int* p = &a;
char* f = (char*)p;
//第一个printf可以不写，因为和你的虚拟内存有关
printf("%p %p %p %d\n", f, &f, &f[0], f[0]);
printf("%02x %02x %02x %02x\n", *(f), *(f + 1), *(f + 2), *(f + 3));
```
